# PerfumeGPT Customer Mobile App

AI-Powered Customer Mobile App for the PerfumeGPT ecosystem. Built with Flutter, this app provides a personalized fragrance shopping experience through AI-driven consultation, interactive quizzes, and a complete e-commerce flow.

## 🚀 Features

- **Guest Mode:** Browse products and consult AI without an initial login.
- **AI Consultation:** Natural language chat interface to get expert fragrance advice.
- **Scent Quiz:** 5-question interactive quiz to discover perfumes matching your personality and occasion.
- **Product Catalog:** Browse a wide range of fragrances with semantic search capabilities.
- **AI Review Summarization:** Quick insights from hundreds of user reviews generated by AI.
- **E-commerce Flow:** Seamless cart management and checkout (authentication required) with support for local payment methods (Momo, VNPay, COD).
- **Profile & Loyalty:** Manage your scent preferences and track your loyalty points (authentication required).

## 🛠 Tech Stack

- **Framework:** Flutter (Material 3)
- **State Management:** Riverpod (with Code Generation)
- **Navigation:** GoRouter
- **Network:** Dio
- **Architecture:** Clean Architecture (Domain, Data, Presentation)
- **Monorepo Management:** Melos

## 📦 Getting Started

1. Ensure you have the Flutter SDK installed.
2. Run `melos bootstrap` from the root directory.
3. Navigate to `apps/customer_app`.
4. Run `flutter run`.

## 🏗 Directory Structure

- `lib/core`: Utilities, constants, theme, and routing configuration.
- `lib/domain`: Entities and Repository interfaces (Business Logic).
- `lib/data`: Repository implementations, Models (DTOs), and data sources.
- `lib/features`: Feature-based UI and providers (Auth, AI Consultation, Store, Order, Profile).
